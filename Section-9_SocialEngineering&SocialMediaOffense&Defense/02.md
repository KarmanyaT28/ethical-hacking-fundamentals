Behavioral Security Controls

1. If you did not request it, do not click on it
2. Never download and run any file you dont trust 100%
3. Never enter sensitive information after following a link or popup
4. Validate the link
5. Minimise personal information disclosure

6. Validate the sender
7. Validate the attachment (Executable, Document, Compression file Extentions, Other)
8. Avoid the obvious threat
9. Vishing & SMShing
